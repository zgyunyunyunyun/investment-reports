可复用的报告模板规范总结
================================================================================

【模板要素】

1. 封面与快照层（必需）
   ├─ 报告标题 + 核心标签
   ├─ 关键数据闪卡（3~6个KPI）
   ├─ 价格/估值当前位置显示
   └─ 导航锚点（粘性导航或快速目录）

2. 估值与历史定位（核心）
   ├─ PB/PE历史区间尺度（分位数）
   ├─ 历史高低点对应股价
   ├─ 内在价值计算（优先PB-ROE法）
   ├─ 多方法估值验证矩阵
   └─ 当前价位在历史上的位置明确

3. 基本面分析（必需）
   ├─ 关键指标5年趋势表
   ├─ 核心压力/优势识别
   ├─ 边际变化信号（重点）
   ├─ 拐点判断（何时有望改善）
   └─ 结构性 vs 周期性区分

4. 安全边际量化（高价值）
   ├─ 三档买入价位建议
   ├─ 历史底部支撑位
   ├─ 最坏情景估算（定量%）
   ├─ 向上空间测算
   └─ 赔率分析（不对称性）

5. 空间与情景分析（高价值）
   ├─ 3年/5年维度情景分析
   ├─ 保守/基准/乐观多情景
   ├─ 每个情景的关键假设
   ├─ 机构预测对标参考
   └─ 年化回报率计算

6. 催化剂管理（高频应用）
   ├─ 正面催化剂清单+时间窗
   ├─ 负面催化剂预警清单
   ├─ 影响程度分级（★数）
   └─ 触发逻辑解释

7. 交易框架（实操价值）
   ├─ 分批买入规则（触发条件+价位+比例）
   ├─ 持有原则（监控指标+频率）
   ├─ 卖出信号系统（5~6个明确信号）
   ├─ 仓位管理建议
   └─ 投资者画像匹配

8. 风险量化（必需）
   ├─ 风险类型分类
   ├─ 当前状态评估
   ├─ 影响程度分级
   ├─ 应对措施明确
   └─ 综合评分卡（多维）

【可视化规范】

数据类：
• 卡片：适合单一KPI或快速数字（PB/PE/股息率/ROE）
• 进度条：适合对比展示（ROE排名、成长率对标）
• 表格：适合多维度对标（12+指标，3+对象）
• 尺度图：适合历史分位（某指标在历史范围的位置）
• 堆栈图：适合收入/成本结构拆解

文字类：
• 加粗高亮：核心判断、关键数字
• 分色分类：Bull/Bear、风险/机会、正面/负面
• 列表缩进：论据体系、操作步骤、注意事项
• 引用框：重要结论、警示、小贴士
• 分隔符：章节分隔，视觉呼吸

【深度判断标准】

低深度报告（简单分析）：
• 只有当前数据快照，无历史对标
• 只有结论，无论据体系
• 只有一种估值方法
• 没有情景分析
• 没有明确的交易规则

中深度报告：
• 有5年历史对标
• 有2~3种估值方法
• 有2个情景分析
• 有概括性交易建议

高深度报告（本次三份的标准）：
• 有5年+历史分位数化
• 有5种估值方法矩阵
• 有3~4个情景分析+定量回报
• 有系统化交易框架（分批买入+持有原则+卖出信号）
• 风险定量化（下行-X% vs 上升+Y%）
• 催化剂监控清单
• 基本面拐点信号识别

【格力电器报告缺失的关键要素】

对比这三份银行报告，格力报告应补充：

1. ✗ 全球/行业对标维度
   格力是否在全球家电企业中估值偏低？与海尔、美的、日本家电厂商对标？

2. ✗ 历史演变5阶段分析
   格力从2008-2025年经历了哪5个关键阶段？当前处于哪个阶段？

3. ✗ Bull/Bear完整论据体系
   看多：超高ROE、现金流、分红是否证明了价值？
   看空：增速放缓、创新不足、新能源转型困难是否意味着衰退？
   两方论据都要有。

4. ✗ 多分类讨论与分层定位
   不同投资者（成长型/价值型/收益型）买格力各自的理由是什么？
   按业务分（空调/小家电/芯片/新能源）的投资逻辑有何不同？

5. ✗ 交易框架系统化
   不是"建议买"而是"分几批买、什么条件下加仓、什么条件下卖"具体规则

6. ✗ 催化剂定量化与时间窗
   不是"可能有利好"而是具体的时间点和预期影响程度

7. ✗ 基本面拐点识别
   哪些边际变化信号表明"最坏已经过去"？（而不只是承认压力存在）

8. ✗ 赔率不对称分析
   最坏情景定量为-X%，向上空间定量为+Y%，为什么值得投？
