两位大师：同源不同路的投资哲学
🧙 沃伦·巴菲特
伯克希尔·哈撒韦 · 奥马哈先知
"我宁愿用合理的价格买入一家优秀的公司，也不愿用便宜的价格买入一家普通的公司。"
  • 定价权：能不断提价而不损失市场份额，是好生意的终极证明。可口可乐从1886年涨价至今，消费者从不离开。
  • 轻资产模型：好生意不需要大量资本投入就能产生高额利润。喜诗糖果25美元投入赚100美元，这才叫好生意。
  • 消费品特许经营：强大品牌在消费者大脑中占据不可替代的位置，形成"心理垄断"。
  • 能力圈边界：不投科技股（早期），不是因为不懂好坏，而是无法预测未来10年技术走向。
  • 财务标准：ROE长期 > 20%，自由现金流持续为正，负债水平合理，资本配置能力强。
  • 管理层：选择像对待自己钱一样管理公司的CEO，避开依靠资本运作而非实际经营的管理层。
🏹 段永平
步步高系 · OPPO/vivo创始精神
"好的商业模式是能长期持续地产生净现金流的东西，而且这种能力不容易被模仿或替代。本分就是做对的事情。"
  • 本分哲学：做对用户真正有价值的事，不骗人不走捷径，长期信任就是最深的护城河。这是段永平对巴菲特"诚信文化"的中国化诠释。
  • "看不懂不投"铁律：20年不碰半导体和汽车，即便周围人大赚特赚。不在能力圈内的机会，就是风险。
  • 重仓苹果逻辑：用户黏性极强，生态系统转换成本极高，硬件只是入口，服务才是引擎。这是他20年来买过的最好生意之一。
  • 重仓茅台逻辑：中国文化属性的消费特许经营，品牌价值无法复制，定价权百年不变。
  • 等待与耐心：2011年大量买入网易，等了7年才兑现。他说："好公司等便宜，便宜等好公司。"
  • 商业模式第一：先判断这是不是一门好生意，再看估值。估值只影响收益率，商业模式决定是否值得持有。

🔗 两人最核心的共同认知

他们的框架看起来不同，但都在回答同一个问题：这门生意10年后还在吗？它的竞争优势还在吗？ 所有的指标——ROE、自由现金流、品牌价值、网络效应——都只是这个问题的验证工具，而非目的本身。

好商业模式的五种护城河（选股底层筛选器）
如何使用护城河框架 护城河决定了一门生意能否长期保持超额利润。一家公司若同时具备2种以上护城河，通常就是值得长期持有的好生意。ROE持续 > 20% 是护城河真实存在的财务佐证。
🧠
品牌溢价
消费者愿意为品牌支付额外费用，且不因价格上涨而流失。品牌等于信任的资产负债表外价值。
茅台 / 爱马仕 / 苹果 / 可口可乐
🌐
网络效应
用户越多，产品越有价值，形成正反馈循环。新进入者面临的不是技术壁垒，而是社会壁垒。
微信 / Visa / Facebook / 美团
🔒
转换成本
用户一旦使用，切换到竞品的代价极高（时间、数据、习惯、迁移成本），形成"被动忠诚"。
微软Office / 苹果生态 / 工业软件
💰
成本优势
凭借规模、流程或资源优势，以竞争对手无法匹配的低成本运营，赢得价格战或获取超额利润。
拼多多 / 宁德时代 / 亚马逊物流
⚖️
特许经营权
牌照、政策保护或垄断资源构成法律层面的准入壁垒，竞争者即便有能力也无法进入。
中国烟草 / 银行牌照 / 机场高速

如何找到"适合自己"的投资赛道？三步筛选法
核心原则：圈子 × 热情 × 边界 好的赛道 = 你能看懂（认知可及）× 你愿意持续研究（有兴趣）× 这门生意本质上是好生意（护城河）。 三者缺一不可。只满足其中一两条的，都可能是陷阱。
稳健收益型
以分红+价值为核心
对行业不需要深入理解，日常生活即能感知，适合时间不多、厌恶波动、追求确定性回报的投资者。
  • 贵州茅台（消费特许经营，最简单）
  • 国有大行H股（分红5%+，底层安全）
  • 可口可乐 KO（百年定价权，极易理解）
  • 招商银行（零售银行，相对易懂）
  • 银行ETF/消费ETF（不选个股的选择）
成长价值型
以护城河+成长为核心
愿意花时间研究，能接受一定波动，追求ROE驱动的长期复利。需要每年持续跟踪公司变化。
  • 腾讯（网络效应+生态，用户即研究员）
  • 宁波银行/成都银行（城商行成长逻辑）
  • 苹果 AAPL（生态锁定，每年换机即调研）
  • Visa/Mastercard（支付网络，最纯粹模型）
  • 微软 MSFT（企业SaaS订阅复利机器）
专业认知型
以认知优势为前提
只在你有真实、独特行业认知的领域参与。若你在这些行业工作，可以构建别人没有的信息优势。
  • 医疗器械（医生/医院从业者）
  • 云计算/SaaS（IT从业者）
  • 新能源/储能（能源工程师）
  • OTC消费医药（品牌可感知部分）
  • ⚠️ 芯片/创新药 = 非专业人士禁区
⚠️ "看不懂"的三种误区 误区一：听到别人推荐就觉得"差不多懂了"。能复述别人的推荐理由 ≠ 真正懂这门生意。
误区二：因为短期涨了很多，觉得"市场认可了，说明是好生意"。涨价和好生意是两件不同的事。
误区三：把"行业景气"等同于"公司护城河"。即便行业增速再快，没有护城河的公司只是帮供应链赚钱。

📌 终极框架：找到适合自己赛道的四问清单

在研究任何一家公司或行业时，对照这四个问题。如果其中任何一个答案是"不确定"，继续研究直到确定，或者放弃。

❶ 这门生意怎么赚钱？

能用一句话说清楚吗？"茅台靠卖高端白酒，消费者主动购买，无需促销，先收款后发货" ——这就是清晰的商业模式。如果解释需要超过三句话，说明你还没真的看懂。

❷ 为什么竞争对手无法复制？

护城河的本质是"你凭什么有这个定价权/客户"。品牌？网络效应？转换成本？成本优势？特许权？这五种护城河缺一种都没有的行业，利润是借来的，不是自己的。

❸ 10年后这门生意还存在吗？

段永平说买股票就是买公司。如果你无法对这门生意的10年后形成基本判断，这不是投资，是赌博。消费品、支付网络、企业软件通常10年后确定性强；芯片技术路径10年后几乎无法预判。

❹ 我的研究比市场平均水平高吗？

你在某个领域有没有认知优势？你是茅台的消费者，是腾讯的用户，是某企业软件的IT员工……日常接触本身就是调研优势。当你的理解深度超过大多数散户，才是合理入场的时机。

❺ 当前估值合理吗？

好公司+合理价格=好投资。段永平反复强调"不贵的时候买好公司"。PE/PB只是参考，更核心的是：当前价格隐含的增长预期，你是否认同？你是否愿意以此价格持有10年？

❻ 最重要：你理解这门生意吗？

这是一切的前提，也是一切的结束。在你真正搞懂一门生意之前，无论多便宜都不值得投。在你真正搞懂之后，无论多贵，至少知道贵在哪里，也知道该不该买。

一句话总结：段永平 × 巴菲特框架的精髓 找到一门有护城河的好生意（消费品/平台/支付是最优先的方向）， 在你真正搞懂它的商业逻辑之后（不是"听说它很好"，而是"我能解释为什么它好"）， 等待一个合理甚至低估的价格买入， 然后长期持有，让时间帮你赚钱。 这不复杂——复杂的只是控制自己"不乱动"的欲望。
