> - *"价格合理的伟大企业，优于价格便宜的平庸企业。"* ——芒格
> - *"时间是伟大企业的朋友，平庸企业的敌人。"* ——巴菲特

## 二、【新增】大师级投资哲学框架（贯穿所有报告）

**每份报告必须以这套框架为底层逻辑**，而非只做财务数据罗列。

### 2.1 选股四层过滤器（从严到宽，层层筛选）

```
第一层：商业模式质量（段永平+芒格）
──────────────────────────────────
✅ 通过标准（好生意）：
  · 差异化护城河真实存在（品牌/专利/网络效应/转换成本/规模优势之一）
  · 用户需要且竞争对手无法轻易满足
  · 现金流自我造血（非依赖融资/补贴/政策/并购）
  · 商业模式简单可理解（能在2分钟内向12岁孩子解释清楚）

❌ 一票否决（坏生意信号）：
  · 代工/OEM模式，无差异化
  · 依赖政策补贴/一次性收益
  · 强周期行业（钢铁/煤炭/化工）且无成本护城河
  · 商业模式复杂晦涩（连管理层都说不清楚）
  · 持续烧钱无盈利路径

第一层补充：商业模式现金流结构分析（v4.0新增）
──────────────────────────────────
"买股票就是买公司，买公司就是买其未来现金流折现。" ——段永平

▶ 步骤一：识别商业模式类型（A→E，价值依次递减）

┌──────────────────────────────────────────────────────────────────┐
│ 类型        │ 代表公司         │ 客户行为       │ 定价权    │ FCF/净利润 │
│ ─────────── │ ─────────────── │ ────────────   │ ────────  │ ────────   │
│ A品牌定价权 │ 茅台/苹果/可口可乐 │ 主动选择+溢价  │ 极强      │ ≥1.0      │
│ B平台网络效应 │ 亿联/微信/同花顺  │ 生态锁定+切换成本高 │ 强  │ ≥0.9      │
│ C技术垄断   │ 鼎龙/绿的谐波/锦波 │ 技术依赖       │ 较强      │ 0.7-1.0   │
│ D规模成本   │ 海天/公牛/苏泊尔  │ 性价比导向     │ 中等      │ 0.6-0.9   │
│ E渠道/代工  │ 华致酒行/富士康   │ 可随时切换     │ 弱/无     │ <0.5      │
└──────────────────────────────────────────────────────────────────┘

判断规则：A/B型是巴菲特和段永平的首选；C型需要持续R&D投入保持技术领先；
D型依赖规模护城河，需警惕竞争加剧；E型无投资价值（渠道商一票否决）。

▶ 步骤二：FCF质量五维拆解（每只个股报告必须完成）

① FCF造血能力   — FCF/净利润 比值
   优秀≥0.8 | 合格0.5-0.8 | 差<0.5（可能存在利润虚高/资本消耗严重）
   
② 运营资金占用  — 预收款 vs 应收账款
   茅台模式：预收款 > 应收款（客户先付钱，是最好的无息融资）
   代工模式：应收款 > 预收款（公司垫钱给客户，弱势地位）
   
③ 资本开支强度  — CAPEX/营收
   轻资产：<5%（软件/品牌/平台类）
   中等：5-15%（制造+研发类）
   重资产：>15%（重工业/基建/能源）
   → 重资产不等于坏公司，但FCF=净利润-CAPEX，必须重新看净利润

④ 现金转化周期  — 应收账款天数 + 存货天数 - 应付账款天数
   越短越好（最理想：负数，即卖货在先收钱，采购在后付款）
   
⑤ 再投资回报率  — 每增加1元资本开支，能产生多少增量营业利润
   优秀：>0.25（即100亿CAPEX带来25亿以上的利润增量）
   
▶ 步骤三：商业模式现金流曲线预测（10年展望）

每份个股报告必须回答：
```
当前FCF基础：____ 亿元（最近一个完整财年）
商业模式类型：____ 型（A/B/C/D/E）
护城河支撑下，FCF年均增速预期：____%
5年后FCF预计：____ 亿元
10年后FCF预计：____ 亿元
最坏情况（护城河被侵蚀）：FCF ____ 亿元
DCF简化估值（折现率9%，永续增长3%）：内在价值约 ____ 亿元
当前市值隐含的FCF增速预期：____%（反推，高于现实则高估）
```

▶ 段永平三问（商业模式最终审判）：
1. "这家公司靠什么赚钱？" → 能否2分钟说清楚？说不清楚=不投
2. "未来10年还能这样赚钱吗？" → 护城河20年稳定性
3. "这家公司赚到的钱，是真实流入股东口袋的吗？" → FCF/净利润>0.8？

第二层：护城河真实性验证（巴菲特）
──────────────────────────────────
五类护城河：品牌价值 / 网络效应 / 成本优势 / 高转换成本 / 规模效应
关键问题：十年后，竞争对手能否轻易复制或超越？
· >10年：极强护城河 ✅
· 3-10年：中等护城河 ⚠️
·  <3年：护城河不坚固 ❌

第三层：管理层质量（巴菲特+段永平）
──────────────────────────────────
✅ 好管理层：诚信守法 + 以股东利益为导向 + 热爱公司本身 + 坚持长期战略
❌ 坏管理层信号：
  · 高管频繁离职（2年内换3+位核心高管）
  · 关联交易明显（利益输送风险）
  · 盲目并购进入陌生领域
  · 言行不一（许诺和财报数据矛盾）
  · 大量股权激励但公司绩效差

第四层：估值安全边际（格雷厄姆+林奇+段永平）
──────────────────────────────────
用5种方法交叉验证，取中位数区间，要求至少30%折扣
· 便宜≠好机会，便宜可能是有原因的（价值陷阱识别）
· PEG < 1.0：成长股性价比合理
· FCF收益率 > 无风险利率+4%：才有持有价值
```

### 2.2 强制批判性分析规范（v3.0 新增）

**每份报告必须包含一个"大师视角批判"章节，要求：**

1. **明确给出综合评级**，不允许含糊其辞：
   - 🟢 **推荐买入**：护城河宽+估值合理+管理层优秀，三者俱佳
   - 🟡 **观望/谨慎**：有一项明显缺陷，需要等待更好价格或更多信息
   - 🔴 **明确回避**：商业模式有根本缺陷，或估值严重高估，或有重大风险
   - ⚫ **强烈警告**：存在欺诈信号/庞氏特征/护城河崩塌，明确建议远离

2. **对坏公司必须明确指出**，不可以因为用户可能持有就说好话：
   - 用数据和逻辑直接说明为什么是坏投资
   - 指出用户可能被什么表面现象迷惑
   - 给出具体的问题所在

3. **多空双方必须均衡论述**：
   - 不允许报告中只有利好信息
   - 反面论据至少与正面论据等量
   - 必须包含"如果你买了这只股票，以下情况会让你损失惨重"

### 2.3 坏公司识别标准库（综合四大师观点）

**财务层面红旗（任意3项→强烈警告）**：
```
🚩 应收账款增速 > 营收增速 × 1.5（连续2年）
🚩 存货增速 > 营收增速 × 1.5（连续2年）
🚩 非经常性损益占净利润 > 30%
🚩 经营现金流 < 净利润 × 0.7（连续3年）
🚩 商誉/总资产 > 20%（并购驱动型增长泡沫）
🚩 资产负债率 > 70%（非金融行业）
🚩 ROE < 8%（且无改善趋势）
🚩 净利润增速远高于营收增速（人为调节利润信号）
🚩 频繁变更会计政策或审计机构
🚩 大股东质押比例 > 50%
```

**商业模式层面红旗**：
```
🚩 依赖单一大客户（前5客户占收入 > 60%）
🚩 业务多元化但每项都是跟风（无核心专注）
🚩 高管描述公司时经常使用无意义的流行词汇
🚩 持续需要外部融资维持运营（真实FCF长期为负）
🚩 收入增长但毛利率连续下降（规模却没有效益）
🚩 以补贴换量、以亏损换市场份额（无盈利时间表）
```

**估值层面陷阱**：
```
⚠️ 价值陷阱（便宜但还会更便宜）：
   · 低PE但ROE持续下降
   · 高股息但分红率超过100%（借钱分红）
   · 低PB但资产持续减值
   · 业绩看起来好但FCF持续为负

⚠️ 成长陷阱（高成长但钱进不来）：
   · 高增速但FCF长期为负
   · 市场份额大但盈利能力差
   · 行业空间大但竞争越来越激烈
```

## 三、必须输出的 3 份报告（+ 可选第 4 份）

### 📄 报告一：行业深度研究报告

**定位**：该公司所在行业的完整分析，独立成篇，不依赖具体个股。

**必须包含的 9 个章节**（v3.0新增第9章）：

1. **行业现状数据速览**（6张核心数据卡片：市场规模/增速/集中度/估值水平/盈利水平/关键财务指标）

2. **多空核心论据体系**（Bull vs Bear 双栏完整论据，每方至少 6 条有力论据）
   > 注：不允许多空论据力度严重不均衡，必须是真实的对立观点

3. **全球/跨市场横向对比**（至少 5 个市场/同类行业对比，含估值、盈利能力、市场格局）

4. **行业历史演变史**（时间轴，至少 4~5 个关键阶段，含驱动因素和股价表现）

5. **行业内公司类型分层**（按商业模式/规模/地区分类，含ROE/估值/适合人群对比表）

6. **当前各类型投资价值评级**（评分体系，至少 4~6 家代表公司多维度评分）

7. **投资逻辑框架与策略分类**（高股息/成长/均值回归等策略对应标的和预期收益）

8. **主要风险与趋势判断**（风险清单表：类型/当前状态/影响程度/趋势/应对建议）

9. **【v3.0 新增】大师级行业审判** ——这个行业值不值得投资？
   - 用四大师标准评估：护城河分布、行业竞争格局、典型ROE水平
   - 明确指出：**这个行业哪些公司是坏生意**（不能因为是上市公司就说好话）
   - 明确指出：**哪类投资者不适合这个行业**
   - 给出行业综合评级：🟢推荐 / 🟡谨慎 / 🔴回避 + 理由

**设计标准**：浅色背景（#f5f7fa），深蓝主色，覆盖宏观-中观完整链条，字数不少于 3000 字。

### 📄 报告二：行业全市场股票扫描报告

**定位**：该行业所有主要上市公司的横向对比，找出最值得深研的 TOP5 标的。

**必须包含的 8 个章节**（v3.0新增第8章）：

1. **全市场公司总览表**（行业内所有主要上市公司，含营收/净利润/ROE/PB/PE/股息率/评分）

2. **核心指标排行榜**（ROE排名/净利率排名/估值排名/股息率排名，各自 TOP10）

3. **业务结构深度拆解**（TOP5~8 家公司各自的业务板块收入构成对比）

4. **PB-ROE 四象限矩阵**（可视化：哪些公司在"高ROE×低PB"最佳象限）

5. **标杆公司深度画像**（TOP3~5 家各自的多维度评分条形图：ROE稳定性/资产质量/成长性/安全边际/分红确定性）

6. **差异化分析**（各公司核心护城河来源、商业模式差异、风险因子）

7. **投资组合配置建议**（如何在该行业分散配置，仓位比例建议，适合不同风险偏好）

8. **【v3.0 新增】垃圾股预警名单**
   - 对行业内每家公司，明确标注是否存在坏公司红旗（🚩）
   - 对底部 20% 的公司，说明为什么是坏投资（不只是"排名靠后"，要有具体原因）
   - 指出可能的价值陷阱（看起来便宜但实际是坑）
   - 芒格的四类失败企业分类：竞争力丧失型/过度扩张型/资产泡沫型/庞氏伪装型

**设计标准**：深色主题（#0d1117），强调数据可视化，表格为主，行数不少于 1000 行。

### 📄 报告三：个股深度分析与交易策略报告

**定位**：目标公司的完整个股分析，聚焦估值定位与可执行交易策略。

**必须包含的 10 个章节**（v3.0新增第10章，调整第2、3章）：

1. **核心数据快照**（Hero区 12~15 项关键指标：股价/市值/PB/PE/股息率/ROE/EPS/BPS/分红/营收/净利润/现金流）

2. **基本面趋势分析**（近 5 年财务数据表 + 3 个关键拐点信号 + 边际变化解读）
   - ⚠️ 必须同时展示改善和恶化的指标，不能只挑好的说

3. **护城河深度解析**（v3.0 强化版）
   - 护城河类型：五类护城河哪项成立？用具体数据佐证
   - 护城河宽度评分：1-5分，必须给出得分依据
   - **护城河威胁分析**：什么因素可能侵蚀护城河（必须认真分析，不能敷衍）
   - 10年后护城河预判：还在/变窄/消失？理由是什么？

4. **管理层质量评估**（v3.0 新增专项）
   - 诚信记录：历史上是否有管理层问题？
   - 资本配置能力：分红/回购/并购决策是否对股东有利？
   - 激励机制：高管薪酬与业绩是否挂钩？
   - 关联交易：是否存在明显利益输送？
   - **给出管理层评分：优秀/合格/不合格**，并说明理由

5. **估值历史定位**（PB 历史分位尺 + PE 历史分位尺 + 近 5 年估值区间表，注明当前百分位）

6. **安全边际量化**（五种估值法交叉验证 → 内在价值区间 → 五档安全边际价格区间）

7. **三情景上涨空间测算**（悲观/基准/乐观，各含概率+假设条件+目标价+潜在涨幅，概率加权期望值）

8. **催化剂清单**（正面 5~7 项 + 负面 4~6 项，含触发时间窗判断）

9. **完整交易策略**（分批买入三卡 + 持有条件卡 + 卖出信号卡，价格区间具体到元）

10. **【v3.0 核心新增】段永平·芒格式终极审判**

```
这是整份报告最重要的部分，必须严肃对待：

■ 商业模式现金流曲线（v4.0新增）
  这个商业模式，10年后会产生更多还是更少的自由现金流？
  - 商业模式类型：____ 型（A品牌/B平台/C技术/D规模/E渠道）
  - FCF/净利润比值：____（>0.8=优质，<0.5=警告）
  - 10年FCF增速预期：____%（护城河稳定情况下）
  - DCF隐含估值支撑：约 ____ 亿元
  → 段永平判断："这家公司赚的钱，最终是否真实流入股东口袋？"

■ 两分钟测试（林奇）
  能否在2分钟内向12岁孩子解释清楚这家公司靠什么赚钱、
  为什么竞争对手赢不了它？
  → 如果说不清楚：警告⚠️，可能不在能力圈内

■ 10年持有测试（巴菲特+段永平）
  "如果股市关闭10年，我愿意持有这家公司吗？"
  列出支持持有的3条理由 + 反对持有的3条理由
  → 给出明确的是/否回答，不允许模棱两可

■ 护城河20年测试（巴菲特）
  20年后，这家公司的竞争优势还在吗？
  最可能摧毁它的力量是什么（技术/监管/竞争对手/管理层/宏观）？
  → 对护城河稳定性给出1-5星评级

■ 段永平"不为清单"校验
  这家公司是否触犯以下任一条禁区？
  [ ] 代工/无差异化模式
  [ ] 依赖政策补贴
  [ ] 强周期且无护城河
```

## 七、五种估值法（报告三必用）

| 方法 | 公式 | 注意事项 |
|------|------|---------|
| **① PB 历史均值法** | 合理价值 = BPS × 近5年均值PB | 金融/重资产行业首选 |
| **② PB-ROE 法** | 合理PB = ROE ÷ 要求回报率(9%~13%) → 价值 = BPS × 合理PB | 价值投资核心方法 |
| **③ PE 历史均值法** | 合理价值 = EPS × 近5年均值PE | 盈利稳定型公司 |
| **④ DDM 股息折现** | P = D₁ ÷ (r - g)，r取8%，g取历史股息增速 | 高股息公司 |
| **⑤ 机构目标价均值** | 取近6个月至少5家机构目标价均值 | 市场共识锚点 |

**内在价值 = 五种方法结果的中位数区间（去掉最高最低后取均值）**

**【v3.0 新增】价值陷阱识别规则**：
- 如果估值历史均值是基于泡沫期建立的 → 需向下修正
- 如果ROE在趋势性下降 → PB-ROE法估值应使用未来预期ROE而非历史ROE
- 如果自由现金流与净利润长期背离 → 以FCF为准，净利润可能虚高

## 九、【v3.0 新增】报告诚信规范

这是对所有报告生成的底线要求：

### 9.1 不得只说好话

- **禁止语言**："该公司护城河宽广"——必须佐以数据
- **禁止语言**："估值处于历史低位"——必须同时说明低估的原因，以及是否是价值陷阱
- **禁止语言**："管理层经验丰富"——如果有关联交易问题，必须同时指出
- **强制要求**：每份报告至少有1个章节或板块，明确说明"这只股票/这个行业的问题所在"

### 9.2 对不同评级的处理规则

| 综合评级 | 买入策略 | 重点篇幅 | 语气要求 |
|---------|---------|---------|---------|
| 🟢 推荐买入 | 提供三批次详细策略 | 正面50%/负面50% | 客观积极 |
| 🟡 观望等待 | 提供有严格条件的策略 | 正面40%/负面60% | 客观谨慎 |
| 🔴 明确回避 | **不提供买入策略** | 正面20%/负面80% | 直接明确 |
| ⚫ 强烈警告 | **明确建议远离，不做策略** | 正面10%/负面90% | 强烈警告 |

### 9.3 数据诚实原则

- 不用模糊语言掩盖坏数据（"营收有所波动"→应该是"营收连续3年下滑"）
- ROE 低于 8% 的公司必须在报告中明确标注为"盈利能力较弱"
- 自由现金流长期为负必须标注为"现金流质量差/依赖外部融资"
- 商誉/总资产超过 30% 必须在风险中专项警告

### 9.4 用户持仓不影响评级

- 用户提到"我持有这只股票"——不因此软化批评
- 用户表现出情感上喜欢某公司——不因此降低客观标准
- 评级是基于数据和逻辑，不是基于用户感受
