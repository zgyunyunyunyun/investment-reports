# 📊 Investment Research Skill v2.0
## 投资研究全流程技能规范

> **核心规则**：每次收到"帮我研究 XXX 股票/行业"的请求，**必须输出至少 3 份 HTML 报告**，在特殊情况下输出第 4 份。  
> 所有报告部署到 GitHub Pages，并同步更新 index.html 导航首页。

---

## 一、触发条件

用户说出以下任意意图，立即按本 Skill 执行：

- "帮我研究 XXX 这只股票"
- "分析一下 XXX 行业"
- "我想投资 XXX，帮我做完整分析"
- "对 XXX 做个深度研究"
- "帮我制定 XXX 的交易策略"

## 二、必须输出的 3 份报告（+ 可选第 4 份）

### 📄 报告一：行业深度研究报告
**定位**：该公司所在行业的完整分析，独立成篇，不依赖具体个股。  
**参考标准**：银行投资价值深度研究报告.html（913行）

**必须包含的 8 个章节**：
1. **行业现状数据速览**（6张核心数据卡片：市场规模/增速/集中度/估值水平/盈利水平/关键财务指标）
2. **多空核心论据体系**（Bull vs Bear 双栏完整论据，每方至少 6 条有力论据）
3. **全球/跨市场横向对比**（至少 5 个市场/同类行业对比，含估值、盈利能力、市场格局）
4. **行业历史演变史**（时间轴，至少 4~5 个关键阶段，含驱动因素和股价表现）
5. **行业内公司类型分层**（按商业模式/规模/地区分类，含ROE/估值/适合人群对比表）
6. **当前各类型投资价值评级**（评分体系，至少 4~6 家代表公司多维度评分）
7. **投资逻辑框架与策略分类**（高股息/成长/均值回归等策略对应标的和预期收益）
8. **主要风险与趋势判断**（风险清单表：类型/当前状态/影响程度/趋势/应对建议）

**设计标准**：浅色背景（#f5f7fa），深蓝主色，覆盖宏观-中观完整链条，字数不少于 3000 字。

---

### 📄 报告二：行业全市场股票扫描报告
**定位**：该行业所有主要上市公司的横向对比，找出最值得深研的 TOP5 标的。  
**参考标准**：银行股深度财报分析与投资图谱2025.html（1378行）

**必须包含的 7 个章节**：
1. **全市场公司总览表**（行业内所有主要上市公司，含营收/净利润/ROE/PB/PE/股息率/评分）
2. **核心指标排行榜**（ROE排名/净利率排名/估值排名/股息率排名，各自 TOP10）
3. **业务结构深度拆解**（TOP5~8 家公司各自的业务板块收入构成对比）
4. **PB-ROE 四象限矩阵**（可视化：哪些公司在"高ROE×低PB"最佳象限）
5. **标杆公司深度画像**（TOP3~5 家各自的多维度评分条形图：ROE稳定性/资产质量/成长性/安全边际/分红确定性）
6. **差异化分析**（各公司核心护城河来源、商业模式差异、风险因子）
7. **投资组合配置建议**（如何在该行业分散配置，仓位比例建议，适合不同风险偏好）

**设计标准**：深色主题（#0d1117），强调数据可视化，表格为主，行数不少于 1000 行。

---

### 📄 报告三：个股深度分析与交易策略报告
**定位**：目标公司的完整个股分析，聚焦估值定位与可执行交易策略。  
**参考标准**：招商银行投资分析与交易策略2026.html（1250行）

**必须包含的 9 个章节**：
1. **核心数据快照**（Hero区 12~15 项关键指标：股价/市值/PB/PE/股息率/ROE/EPS/BPS/分红/营收/净利润/现金流）
2. **基本面趋势分析**（近 5 年财务数据表 + 3 个关键拐点信号 + 边际变化解读）
3. **业务结构深度拆解**（各业务板块收入/利润贡献、核心护城河来源、竞争优势量化）
4. **估值历史定位**（PB 历史分位尺 + PE 历史分位尺 + 近 5 年估值区间表，注明当前百分位）
5. **安全边际量化**（五种估值法交叉验证 → 内在价值区间 → 五档安全边际价格区间）
6. **三情景上涨空间测算**（悲观/基准/乐观，各含概率+假设条件+目标价+潜在涨幅，概率加权期望值）
7. **催化剂清单**（正面 5~7 项 + 负面 4~6 项，含触发时间窗判断）
8. **完整交易策略**（分批买入三卡 + 持有条件卡 + 卖出信号卡，价格区间具体到元）
9. **风险评分矩阵**（6~8 类风险，每类含名称/评分条/风险等级文字说明）

**设计标准**：深色主题（#0d1117），重点突出买入价格区间，所有策略量化到具体数字，行数不少于 1200 行。

### 📄 报告四（条件触发）：投资理念/认知框架报告
**触发条件**（满足任意一条时生成）：
- 该行业是用户首次研究的新领域
- 该行业有重大认知争议（如"银行股是否值得投资"、"消费股是否还有未来"）
- 用户明确提出想了解该行业的底层投资逻辑

**定位**：该行业的底层商业逻辑、投资哲学、认知框架。  
**参考**：巴菲特最新持仓深度解读2025.html + 好商业模式投资框架与赛道选择.html

**必须包含**：
- 行业商业模式本质解析（用一个类比说清楚）
- 主流投资派系对该行业的不同看法
- 适合投资该行业的投资者画像
- 与其他行业的横向比较（为什么选/不选这个行业）

## 四、数据收集标准

### 4.1 并发数据收集（并行执行，不要串行）

**启动 2~3 个并行 Agent 同时收集**：

```
Agent A（基本面数据）：
- 目标公司近5年：营收/净利润/ROE/EPS/BPS/每股分红/经营现金流
- 业务板块收入拆解（最新年报）
- 管理层变化/股东结构

Agent B（估值历史数据）：
- 近5年PB区间：历史最高/最低/均值/当前值/百分位
- 近5年PE区间：同上
- 机构目标价（至少5家券商：券商名/评级/目标价）
- 股价历史高低点

Agent C（行业数据）：
- 行业市场规模、增速、集中度
- 主要竞争对手财务对比
- 行业政策变化
- 行业估值水平横比
```

### 4.2 数据来源优先级

| 类型 | 首选来源 | 备用来源 |
|------|---------|---------|
| 年报财务数据 | 巨潮资讯、新浪财经年报页 | 东方财富、同花顺 |
| 估值历史 | 理杏仁（lixinger.com） | 东方财富choice |
| 机构目标价 | 东方财富研报页 | 各大券商官网 |
| 行业数据 | 前瞻产业研究院、奥维云网 | 行业协会官网 |
| 竞争格局 | 公司年报、券商行业研报 | 财经媒体 |

### 4.3 数据质量要求

- 所有财务数据必须标注来源和截止日期
- 估值分位数必须标注参考周期（近3年/近5年/近10年）
- 机构目标价必须标注报告日期（只使用近 6 个月的数据）
- 不确定的数据标注"约"或"估算"，不能伪造精确数字

## 六、五种估值法（报告三必用）

| 方法 | 公式 | 注意事项 |
|------|------|---------|
| **① PB 历史均值法** | 合理价值 = BPS × 近5年均值PB | 金融/重资产行业首选 |
| **② PB-ROE 法** | 合理PB = ROE ÷ 要求回报率(9%~13%) → 价值 = BPS × 合理PB | 价值投资核心方法 |
| **③ PE 历史均值法** | 合理价值 = EPS × 近5年均值PE | 盈利稳定型公司 |
| **④ DDM 股息折现** | P = D₁ ÷ (r - g)，r取8%，g取历史股息增速 | 高股息公司 |
| **⑤ 机构目标价均值** | 取近6个月至少5家机构目标价均值 | 市场共识锚点 |

**内在价值 = 五种方法结果的中位数区间（去掉最高最低后取均值）**

## 七、完整交易策略标准格式（报告三）

### 7.1 分批买入（三批次）

```
第一批（30~40% 仓位）：[具体价格区间，元]
  → 触发条件：当前估值/PB/PE在什么水平
  → 此时对应股息率/PB/PE是多少

第二批（20~30% 仓位）：[具体价格区间，元]（比第一批低 10~15%）
  → 触发条件：[具体量化条件]

第三批（20~30% 仓位）：[具体价格区间，元]（比第一批低 20~30%）
  → 触发条件：极端低估区，PE/PB接近历史最低
```

### 7.2 持有原则（列举 4~6 条具体可验证的条件）

```
✅ 继续持有，当以下条件全部成立：
1. ROE 维持在 XX% 以上（指定数字）
2. 核心护城河（[具体描述]）无实质性恶化
3. PB 未超过历史均值的 1.3 倍（XX× 以下）
4. 分红持续稳定或增长
5. [该公司特有的关键指标]不发生实质恶化
```

### 7.3 卖出信号（分三级）

```
第一级：部分减仓（减 30%）
  → 触发：股价达到 XX 元，PE/PB 超过 XX

第二级：大幅减仓（减 50%）
  → 触发：股价达到 XX 元，接近历史估值高位

第三级：清仓
  → 触发（任一条件）：
    · ROE 连续 3 年低于 XX%
    · 核心护城河出现[具体描述的实质损伤]
    · 管理层出现重大诚信危机
    · 监管政策根本性改变行业格局
    · 发现更优质替代标的（机会成本）
```
