【第一梯队 - 核心配置（权重30-40%）】

1. 中际旭创 (300308) - 评级 ★★★★★
   • 最新股价：851元｜总市值：8163-9000亿元｜PE：23-25倍｜毛利率：50%
   • 2025年营收：382.4亿（+60.25%）| 净利润：107.97亿（+108.78%）
   • Q1 2026单季利润57.35亿元，已超2024年全年总额
   
   投资逻辑：
   ✓ 1.6T光模块NVIDIA独家供应商，供应垄断地位确立
   ✓ 全球超大规模云厂商AI资本开支1186亿美元同比+64%，订单锁定至2027年
   ✓ 利润爆发周期启动，2026年全年净利润预计300-350亿元

2. 新易盛 (300502) - 评级 ★★★★★
   • 最新股价：589元｜总市值：5000-5500亿元｜PE：19倍｜毛利率：58%
   • 2025年净利润：94-99亿（+231-249%）- 业界最高增速
   
   投资逻辑：
   ✓ LPO技术成本领先，毛利率58%远超中际旭创50%
   ✓ PE 19倍远低于中际旭创23-25倍，性价比最优
   ✓ "小盘股涨法"最优标的：低PE+超高增速组合

   • 2025年营收：1647.82亿（+43.25%）| 净利润：24.13亿（仅+5.20%）
   • 现金流：54.53亿（+7183.71%）- 核心改善指标
   
   投资逻辑：
   ✓ AI服务器龙头，国内市占率52.3%，全球第二仅次DELL
   ✓ 利润拐点即将到来：AI占比从30-35%升至50%+后，毛利率从15-16%升至20%+
   ✓ 一旦利润加速释放，净利润增速有望从5%加速至30%+（2027年）

   • Q1 2026营收：40.34亿（+68.06%创单季新高）

   ✓ 国产替代龙头，美国管制升级驱动需求刚性
   ✓ 海光DCU进口替代方案，市场空间超500亿美元
